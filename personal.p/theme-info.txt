INSTALLATION STEPS:
==================

1. Upload theme files to /wp-content/themes/autodealer-pro/
2. Activate the theme in WordPress admin
3. Go to Settings > Permalinks and save (to refresh URL structure)
4. Start adding cars via the Cars menu in admin
5. Create navigation menus under Appearance > Menus
6. Customize settings under Appearance > Customize
